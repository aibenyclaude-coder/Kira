# Kira

> **Where agents shine.**

人間の知見をエージェントに渡す仕組み。
エージェントが同じ壁に二度ぶつからない世界を作る。

---

## Core Idea

作業の瞬間、エージェントがキーワードでindexを引く。
必要な指示が降ってくる。ミスらない。同じ苦しみを繰り返さない。

- **Runtime fetch** — 事前ロードせず、必要な一枚だけ
- **Read all** — 複数候補を全部読み、エージェントが文脈で選ぶ
- **Upfront disclosure** — エージェントは最初に選択理由を宣言
- **Multi-AI** — Claude / GPT / Gemini / open models すべてで動く

## Model

- **コミュニティSkill** — 無料。初期貢献者は永続無料枠
- **ベンダーSkill** — 審査制。金ではなく品質証明で入場
- **指示品質を競う市場** — AdWordsではなく品質のオークション
